# 📈 Walmart Weekly Sales Forecasting

A Machine Learning web application that predicts Walmart weekly sales using a Random Forest Regression model. The application is built with Streamlit and provides an interactive interface for real-time sales forecasting.

## 🚀 Features

- Predict Walmart weekly sales
- Interactive Streamlit dashboard
- Real-time forecasting
- Batch prediction using CSV upload
- Download prediction results
- Visual analytics with Plotly

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest Regressor
- Streamlit
- Plotly

## 📊 Model Features

- Week
- CPI
- Unemployment
- Store Size
- Store Type
- Department
- Store ID

